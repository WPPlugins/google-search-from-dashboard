Allows you to search the web from your dashboard.

== Description ==

This plugin adds a web search box that allows you to search the web. You can set up the plugin to use Google, Yahoo, Bing, and Ask.com. You can also set it up to ask which search engine you use when you search.

**UPGRADE WARNING**
You will have to reactivate the plugin when upgrading to 2.0. Because the new version ships under a different name.

== Installation ==

1. Upload `web-search.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Search the internet all you want from your dashboard.
